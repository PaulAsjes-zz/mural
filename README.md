# Mural

Customisable tiled presentation. Items will automatically realign to appropriate columns based on browser size. Drag and drop to change order.

## Getting Started

Download the [production version][min] or the [development version][max].

[min]: https://raw.github.com/PaulAsjes/mural/master/dist/jquery.mural.min.js
[max]: https://raw.github.com/PaulAsjes/mural/master/dist/jquery.mural.js

In your web page:

```html
<script src="jquery.js"></script>
<script src="dist/jquery.mural.min.js"></script>
<script>
jQuery(function($) {
  $("#container").mural({
  	// options...
  	itemSelector: ".mural-item",
  	speed: 250
  });
});
</script>
```
